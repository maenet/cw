# go-chatwork

Go client library for the Chatwork API

## Installation

```
$ go get https://github.com/maenet/go-chatwork
```

## Usage

Please check the [example cli](https://github.com/maenet/go-chatwork/blob/main/cmd/sendmessage/main.go).
